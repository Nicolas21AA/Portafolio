Proyectos

Proyecto 1

Este proyecto consiste en el desarrollo de un script en Python que establece conexiones simultáneas a dos bases de datos distintas, PostgreSQL y MySQL, permitiendo ejecutar consultas y extraer datos de ambas plataformas de manera eficiente. A través de este enfoque de multiconexión, el script ofrece una solución versátil para integrar y acceder a información distribuida en diferentes sistemas de gestión de bases de datos, lo cual es común en aplicaciones que manejan múltiples fuentes de datos.

Proyecto 2

Este proyecto consiste en el desarrollo de una aplicación de consola en C++ que implementa un sistema CRUD (Crear, Leer, Actualizar y Eliminar) para gestionar una base de datos de usuarios, utilizando Oracle como sistema de gestión de bases de datos. La aplicación permite a los usuarios realizar diversas operaciones sobre los registros, comenzando por la creación de nuevos usuarios mediante la recopilación de información como nombre, apellido, correo electrónico y número de teléfono. Además, proporciona funcionalidades para leer y visualizar todos los registros existentes, lo que facilita la consulta y búsqueda de información específica. Los usuarios pueden actualizar datos de registros existentes para mantener la información relevante y actualizada, así como eliminar registros de la base de datos cuando sea necesario, asegurando una gestión eficiente de los datos.

Proyecto 3

Este proyecto consiste en el desarrollo de una aplicación web utilizando JavaScript que establece una conexión con un clúster de CockroachDB para gestionar y manipular datos de manera eficiente. La aplicación permite a los usuarios interactuar con la base de datos a través de un conjunto de operaciones CRUD (Crear, Leer, Actualizar y Eliminar), facilitando la gestión de información en tiempo real. Al utilizar CockroachDB, se aprovechan sus características de escalabilidad, resiliencia y consistencia, lo que resulta ideal para aplicaciones que requieren un alto rendimiento y disponibilidad. La conexión se realiza mediante un cliente de base de datos que se integra con el backend de la aplicación, permitiendo a los desarrolladores ejecutar consultas SQL y recibir respuestas en formato JSON para su posterior visualización.
